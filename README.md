# CFST One-line Installer

An easy, fast, and smart way to install and run the CloudflareSpeedTest tool on Termux.

## Installation
To install and run the scanner for the first time, simply copy and paste this command into your Termux:

```bash
curl -sSL https://raw.githubusercontent.com/alinasrollahzadeh405-sudo/cfst-oneline-installer/master/cfsti.sh -o cfsti.sh && chmod +x cfsti.sh && bash cfsti.sh

## Usage
​After the first installation, you can run the scanner anytime by simply executing:ba bash cfsti.sh

### Note: The script will automatically ask you if you want to update the scanner to the latest version.
​Features
​Smart Installer: Installs all requirements automatically.
​Auto-Update: Always fetches the latest version.
​User-Friendly: Asks you before performing an update.
​Clean: Keeps everything organized in a "cfst" folder.
